Placeholder

实例参数

    {
        element: $('input'),        // 输入框元素，注意 placeholder 的属性为 data-placeholder
        nativeFirst: false          // 可选，是否优先使用原生的 placeholder
                                    //     默认为 false ( 为统一浏览器体验考虑 )
    }

静态参数

    Placeholder.template 是包装输入框元素的模版，默认的应该够用了
    如果需要修改，只需注意用来显示 placeholder 的元素为 div 即可

使用方法

    // 单个初始化
    var placeholder = new Placeholder(options);
    placeholder.getPlaceholder();      // 获取 placeholder
    placeholder.setPlaceholder('new'); // 设置 placeholder
    placeholder.dispose();             // 销毁对象

    // 批量初始化
    Placeholder.init();
