鼠标滚轮

实例参数

    {
        element: $('div'),                       // 可选，监听滚动的元素
                                                 //     默认是 $(document)
        onScroll: function (data) {}             // 可选，滚动触发，返回 false 可阻止滚动
    }

使用方法

    var wheelScroll = new WheelScroll(options);
    wheelScroll.dispose();
滚动鼠标滚轮，打开控制台看输出
