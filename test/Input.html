输入框

实例参数

    {
        element: $('input'),                // 输入框元素
        longPress: false,                   // 可选，长按是否触发　onChange
                                            //     默认为 false
        onChange: function () {},           // 可选，相当于 html5 的 oninput 事件
        onEnter: function () {},            // 可选，按下回车触发
        onLongPressStart: function () {},   // 可选，长按开始
        onLongPressEnd: function () {},     // 可选，长按结束
        keyEvents: {                        // 可选，配置 keyCode => event handler
            '38': function () {
                console.log('up');
            }
        }
    }

使用方法

    var input = new Input(options);
    input.dispose();  // 销毁对象
